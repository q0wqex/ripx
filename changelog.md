# Changelog

## [2.2.0] - 2026-02-02
### Добавлено
- **Кастомные темы**: теперь доступно 4 цветовые схемы (Кристалл, Простая, Полночь, Закат).
- **Оптимизированная тема**: «Простая» тема без анимаций и размытия для слабых устройств.
- **Векторные иконки**: переход с Emoji на профессиональный сет Lucide Icons (shadcn style).
- **Переключатель тем**: удобное меню в футере на каждой странице.

### Изменено
- Весь CSS переведен на переменные для мгновенной смены тем без перезагрузки.
- Оптимизирован радиус скругления и анимации в зависимости от выбранного режима.
- Футер пересобран для удобного доступа к настройкам.

---


## [2.1.0] - 2026-01-31
### Добавлено
- Анимированный CSS-котик для индикации загрузки.
- Признательность Remnawave за вдохновение в дизайне.
- Модальное окно обновлений с поддержкой синхронизации через GitHub.
- Кастомный премиальный дизайн скроллбара и эффекты Glassmorphism.
- Серверное кэширование и автоматический поиск файла ченджлога.

### Изменено
- Унифицирован дизайн кнопок: теперь все кнопки действий выглядят одинаково.
- Улучшена верстка шапки альбомов, исправлены конфликты элементов.
- Обновлена визуальная стилистика текста в интерфейсе.

### Удалено
- Удален FFmpeg и функционал конвертации видео (оптимизация веса образа).
