# Changelog

## [2.1.0] - 2026-01-31
### Добавлено
- Анимированный CSS-котик для индикации загрузки.
- Признательность Remnawave за вдохновение в дизайне.
- Модальное окно обновлений с поддержкой синхронизации через GitHub.
- Кастомный премиальный дизайн скроллбара и эффекты Glassmorphism.
- Серверное кэширование и автоматический поиск файла ченджлога.

### Изменено
- Унифицирован дизайн кнопок: теперь все кнопки действий выглядят одинаково.
- Улучшена верстка шапки альбомов, исправлены конфликты элементов.
- Обновлена визуальная стилистика текста в интерфейсе.

### Удалено
- Удален FFmpeg и функционал конвертации видео (оптимизация веса образа).
